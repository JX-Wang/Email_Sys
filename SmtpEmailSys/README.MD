## A Simple Email Sysytem (base Python 2.7)
*注： 只适用于阿里云企业邮箱*
#### Features
  * 邮件阅读
  * 邮件接收与发送
  * 邮件的删除
  * 黑/白名单过滤
  * 附件添加
  * 邮件信息加密（AES）
